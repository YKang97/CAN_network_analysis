## Requirements

- Python ≥ 3.9
- See `requirements.txt` for package dependencies

## How to Reproduce

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Place your data file at `./data/CAN.xlsx`
4. Open `CAN_network_analysis.ipynb` in Jupyter
5. Run all cells sequentially (Cell 1 → Cell 6)

## Data Availability

Raw data are available upon reasonable request from the corresponding
author. The data file is not included in this repository to protect
participant privacy.
